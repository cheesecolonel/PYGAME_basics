2. **what is pygame?**
    1. pygame 提供 display,sound,image,text,event 等模組
    2. pygame可製作2d遊戲
    3. pygame偵測使用者鍵盤、搖桿與滑鼠
    4. pygame可提供內建game objects來做遊戲
